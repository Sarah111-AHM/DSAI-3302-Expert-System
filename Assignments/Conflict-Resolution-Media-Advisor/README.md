# Conflict Resolution in a Media Advisor Expert System

---

## Overview

You are tasked with designing a Conflict Resolution Mechanism for a Media Advisor Expert System.  
The goal is to resolve conflicts when multiple rules apply at the same time.

**Repository:** [GitHub Link](git@github.com:mtahrawi/Expert-systems-26.git)

---

## Required Tasks

1. Define a scenario where conflicting rules might arise
2. Propose a Conflict Resolution Mechanism for the scenario
3. Describe the method(s) used to resolve conflicts (e.g., priority, recency, specificity)
4. Explain why your chosen method is appropriate
5. Create a sample set of rules and demonstrate how your mechanism resolves conflicts between them
6. Upgrade your mechanism using the attached Python work

---

## Notes

- Watch for conflicting rules in real-world examples  
- Focus on clarity and reasoning behind your conflict resolution method  
- Use the provided Python work to test and demonstrate your solution
